# Code Of Conduct

## Our Standard

Contributors and maintainers should help keep this project:
- respectful
- technically constructive
- privacy-aware
- safety-conscious

Expected behavior includes:
- using professional, respectful language
- focusing criticism on behavior, code, contracts, and process
- respecting privacy, IP, and security boundaries
- avoiding harassment, abuse, or deceptive behavior

Unacceptable behavior includes:
- harassment or personal attacks
- doxxing or disclosure of private information
- publishing secrets, credentials, or sensitive user/project data
- abusive, discriminatory, or threatening conduct

## Enforcement

Maintainers may remove, edit, or reject contributions and participation that violate this code of conduct.

## Reporting

For ordinary conduct concerns, open an issue only if doing so will not expose sensitive information.

For security-sensitive matters, use the process in:
- [SECURITY.md](SECURITY.md)
